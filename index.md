# Claude Code

An AI-powered coding agent that operates directly in your terminal. This system teaches you how to build the architecture behind tools like Cursor, GitHub Copilot, and Claude Code itself — from the agent loop to tool execution, context management, and streaming LLM communication.

By completing this system, you will:

- Build a fully functional AI coding agent CLI
- Implement the agent loop (think → act → observe)
- Create a plugin/tool system for extensibility
- Handle streaming LLM responses
- Manage conversation context and memory
- Build a review and feedback system
